## 1. 배포 승인 기준 (Sign-off Criteria)
최종 배포를 위해 반드시 통과해야 하는 필수 관문입니다.

### 1-1. 기능적 무결성 (Functional)
- **초음파 분석:** 가이드라인에 맞는 초음파 사진 업로드 및 분석 성공률 100%
- **AI 파이프라인:** 이미지 생성 엔진 연동 및 결과물(3장 패키지) 생성 정합성 확인
- **결제 시스템:** 5,900원 단건 결제(토스/카카오페이 등) 및 결제 완료 후 결과 페이지 리다이렉트 성공률 100%
- **이미지 저장:** 생성된 고해상도 이미지의 모바일 갤러리 저장 기능 정상 작동 (iOS/Android)

### 1-2. 데이터 및 보안 (Privacy & Security)
- **데이터 파기:** 생성 완료 후 24시간 내 원본 데이터 및 결과물 자동 파기 로직 검증 (개인정보 보호)
- **법적 고지:** 이용약관 및 개인정보 처리방침 링크 연결 및 내용 확인

### 1-3. 호환성 및 환경 (Compatibility)
- **브라우저:** iOS Safari, Android Chrome, Samsung Internet 최적화 확인
- **반응형:** iPhone 13~16, Galaxy S21~24 시리즈 주요 해상도 레이아웃 깨짐 없음

## 2. 개발자가재 셀프 테스트 체크리스트 (Developer Checklist)
개발 단계에서 스스로 검증하여 QA 루프를 단축하기 위한 리스트입니다.

- [ ] **이미지 예외 처리:** 10MB 초과 파일 또는 비이미지 파일 업로드 시 적절한 에러 메시지 노출 여부
- [ ] **중복 방지:** 결제 버튼 또는 생성 시작 버튼의 더블 클릭(Double-tap) 방지 로직 적용 여부
- [ ] **로딩 상태:** "정밀 분석 중" 상태에서 브라우저 이탈 후 재진입 시 상태 유지 여부
- [ ] **API 에러 핸들링:** AI 엔진 또는 결제 API 응답 지연/오류 시 Error Boundary 작동 및 사용자 안내
- [ ] **클린업:** 결과물 저장 후 불필요한 캐시 또는 세션 데이터 정리 로직 확인

## 3. 사용자 경험 및 감성 품질 테스트 (Emotional QA)
'럭셔리 UI'에 걸맞은 감성적 마감도를 체크합니다.

### 3-1. 애니메이션 부드러움 (Smoothness)
- **Glow Curve:** 골드 버튼 및 강조 요소의 Glow 애니메이션(`cubic-bezier(0.45, 0.05, 0.55, 0.95)`)이 프레임 드랍 없이 부드럽게 작동하는가?
- **Transition:** 페이지 전환 및 모달 팝업 시 페이드 인/아웃 처리가 고급스러운가?

### 3-2. 인터랙션 피드백 (Interaction)
- **Tactile Feeling:** CTA 버튼 Hover 시 `scale(1.02)`와 Shadow 변화가 '클릭하고 싶은' 반응성을 주는가?
- **Loading UX:** "정밀 분석 중" 문구가 단순 대기가 아닌 '특별한 가치를 만드는 과정'으로 느껴지도록 시각적 피드백이 충분한가?

### 3-3. 시각적 정밀도 (Visual Precision)
- **Color Consistency:** 골드 그라디언트(#A67C00 ~ #FFBF00)가 저사양 디스플레이에서도 밴딩(Banding) 현상 없이 깔끔하게 출력되는가?
- **Typography:** H1(64px) 및 Subtext(18px)의 가독성과 자간(-0.04em)이 프리미엄 무드를 유지하는가?
