# 헬로베베(Hello Bebe) MVP 요구사항 정의서 (최종 확정)

## 1. 서비스 정의
- **핵심 가설**: "우리 아기의 진짜 얼굴(실사) 3장을 5,900원에 만나는 경험"
- **데드라인**: 2026-02-11 (7일 이내 출시)

## 2. 기능 명세 (Strict MVP)
- **실사화 엔진**: 오직 극사실주의(Realism) 모델만 사용. 지브리 등 아트 스타일 배제.
- **결과물 구성**: 다양한 표정/각도의 실사 이미지 총 3장 제공 (단일 패키지).
- **결제 동선**: 5,900원 단건 결제 유도 (업로드 후 결제 즉시 발생).
- **영상/인증서**: 전면 제외. 오직 고해상도 이미지 다운로드에 집중.

## 3. 사용자 여정 (The 5-Step Logic)
1. [Landing] 실사화 비포/애프터 강조 (신뢰도).
2. [Upload] 초음파 사진 업로드 (가이드라인 준수 확인).
3. [Payment] 5,900원 결제 (간편결제 우선).
4. [Processing] "정밀 분석 중" 로딩 UI (기대감 고취).
5. [Result] 3장의 실사 사진 확인 및 전체 저장.

## 4. 기술 제약
- 이미지 생성 대기 시간 최대 1분 이내 목표.
- 모바일 갤러리 접근 권한 및 이미지 저장 기능 필수.
